---
title: Keyboard Shortcuts
description: Full keybind reference for all Yokai TUI views.
---

## Dashboard View

| Key       | Action                              |
|-----------|-------------------------------------|
| `n`       | Deploy new service (launch wizard)  |
| `s`       | Stop selected service               |
| `r`       | Restart selected service            |
| `l`       | View logs for selected service      |
| `d`       | Open device manager                 |
| `g`       | Open Grafana dashboard              |
| `c`       | Open AI tool configuration          |
| `j` / `k` | Navigate down / up in service list |
| `Tab`     | Switch between tab bar views        |
| `Enter`   | View service details                |
| `?`       | Show help overlay                   |
| `q`       | Quit Yokai                          |

## Device Manager

| Key       | Action                    |
|-----------|---------------------------|
| `a`       | Add new device            |
| `x`       | Remove selected device    |
| `e`       | Edit device configuration |
| `t`       | Test device connectivity  |
| `j` / `k` | Navigate down / up       |
| `Esc`     | Return to dashboard       |

## Deployment Wizard

| Key       | Action                    |
|-----------|---------------------------|
| `j` / `k` | Navigate options         |
| `Enter`   | Select / confirm         |
| `Esc`     | Cancel / go back         |
| `Tab`     | Next field               |
| `/`       | Search models            |

## Log Viewer

| Key       | Action                    |
|-----------|---------------------------|
| `j` / `k` | Scroll down / up         |
| `G`       | Jump to bottom            |
| `g`       | Jump to top               |
| `f`       | Toggle follow mode        |
| `/`       | Search in logs            |
| `Esc`     | Return to dashboard       |

## AI Tool Configuration

| Key       | Action                    |
|-----------|---------------------------|
| `j` / `k` | Navigate tool list        |
| `Space`   | Toggle tool selection     |
| `Enter`   | Toggle selection / confirm all |
| `Esc`     | Cancel                    |

## Global

| Key       | Action                    |
|-----------|---------------------------|
| `?`       | Toggle help overlay       |
| `q`       | Quit (from any view)      |
| `Ctrl+c`  | Force quit                |
