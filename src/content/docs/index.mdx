Single binary, no Python, no npm. SCP to any machine and go.

</div>

Live GPU utilization, VRAM, temperature, power draw. Sparklines and stream charts.

</div>
<div class="feature-card">

### Deploy in Seconds

Guided wizard: pick workload type, device, image, model вЂ” done.

</div>
<div class="feature-card">

### Fleet-Wide Monitoring

Prometheus + Grafana auto-deployed. One key opens your dashboard.

</div>
<div class="feature-card">

### AI Tool Integration

Auto-configure VS Code Copilot, OpenCode, and OpenClaw with one keystroke.

</div>

### Tailscale Native

Discover and connect to GPU machines across your Tailscale mesh automatically.
